# Idle Farm - Silo

Genre: Idle farming simulation
Player: Click/tap interactions, automatic resource generation
Art style: 2D pixel art, tile-based
UI: uGUI (Canvas), pixel-perfect scaling
Pipeline: URP (Universal Render Pipeline)
Input: Input System (new)

UI conventions:
- All UI anchored to Canvas with pixel-perfect scaling
- Use TextMeshPro for text
- Buttons use 9-sliced sprites for rounded corners
- HUD elements anchored to screen edges
- Tooltips anchored near interactable elements
